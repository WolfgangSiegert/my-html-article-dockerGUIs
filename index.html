Conclusion

GUI Pros Cons
Kitematic
  • Officially supported by Docker
  • Very clear and simple interface
  • Not a web app
DockStation
  • Well-designed interface
  • More powerful than Kitematic and has more features
  • Supports remote SSH access
  • Not open source
  • Not web-based
Portainer
  • Web-based - simple setup (it runs in a container)
  • Easy to navigate
  • User management
  • Couldn't figure out how to change the restart policy on containers!
